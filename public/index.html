nodecdn

NodeCDN — fast javascript cdn service used npm base for packages

Example of use

nodecdn.com/:packageName@:packageVersion/:file
nodecdn.com/:packageName/:file — latest package version
nodecdn.com/react@18.0.0/react.js

Cache suite
